# Gully Captain

Frontend-only gully sports utility built with plain HTML, CSS, and JavaScript.

## Files

- `index.html`
- `style.css`
- `script.js`
- `site.webmanifest`
- `.nojekyll`

## GitHub Pages

1. Push this project to a GitHub repository.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your main branch and the `/(root)` folder.
6. Save and wait for the site to publish.

If your repository name is:

- `username.github.io`, the site URL will be `https://username.github.io/`
- anything else, the site URL will be `https://username.github.io/repository-name/`

## Notes

- The app is fully static and works on GitHub Pages.
- All state is temporary and resets on refresh.
- There is no backend, database, authentication, or browser storage.
